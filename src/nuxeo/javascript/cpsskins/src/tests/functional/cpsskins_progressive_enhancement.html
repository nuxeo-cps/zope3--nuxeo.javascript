CPSSkins: progressive enhancement

Standard HTML widgets located on the page can be replaced by Ajax-enhanced widgets.

If the Ajax library is not supported by the browser or if javascript is disabled, the original HTML widgets will be used for compatibility reasons.

This is the opposite of graceful degradation.
