==========================
CPS Third party js library
==========================

This package contains zope 3 packages that embed third party js library
into resource directories.

so far:

  - ctal: client-side templating language

  - json: lightweight data-interchange format
 
  - prototype: JavaScript framework.

  - scriptaculous: JavaScript framework on the top of prototype, that brings  
    enhancments, such as effects.

  - calendar: Javascript Calendar widget

  - cpsskins: Javascript Framework on the top of prototype, that brings  
    MVC oriented features, among other things
